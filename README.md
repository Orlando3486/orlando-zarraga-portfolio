# Orlando Zarraga - Fullstack Portfolio

Portafolio profesional desarrollado con arquitectura fullstack moderna.

## 🚀 Tech Stack

Frontend:
- Next.js (App Router)
- TypeScript
- TailwindCSS
- SSR

Backend:
- NestJS
- TypeScript
- Arquitectura modular
- REST API

## 🏗 Arquitectura

Monorepo:

/front → Aplicación Next.js  
/back → API backend con NestJS  

Comunicación vía REST API.

## ⚙️ Instalación

### Backend

cd back  
npm install  
npm run start:dev  

Servidor en:
http://localhost:4000

### Frontend

cd front  
npm install  
npm run dev  

Aplicación en:
http://localhost:3000

## 🔌 Endpoints

GET /projects  
GET /projects/featured
